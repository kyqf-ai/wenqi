# 问期 · 婚期择吉考据工具

**一份把"算法"摊开给你看的婚期择吉工具**

## 📖 项目简介

问期是一个透明化的婚期选择工具。与传统的黑盒式择吉方法不同，问期将每一个日期的吉凶评分详细拆解为三个层次：

- **通书历法层**：建除十二神、黄道黑道、二十八宿、阴阳不将、天赦日、月德/天德日等
- **生肖太岁层**：生肖冲煞、大利月、杨公忌、三娘煞等民俗要素
- **八字呼应层**：候选日柱与双方四柱的合冲刑害、化解机制

**每一分都来自可查证的依据，让择吉不再是神秘的仪式，而是可推敲的过程。**

## ✨ 核心特性

### 三层互证算法
```
总分 = 通书层(满40) + 生肖层(满25) + 八字层(满35)
```
- 逐项加减，来源明确
- 支持严格模式（硬性排除）和宽松模式（扣分处理）
- 民俗附加项（杨公忌、三娘煞等）可选启用

### 双视图展示
- **月历视图**：直观浏览整个月份，一眼看出最优日期
- **榜单视图**：按分数从高到低排列，便于精确比较

### 灵活的参数配置
- 自定义查询范围（3个月 / 12个月 / 今年 / 明年 / 自定义）
- 双方四柱八字输入（出生日期 + 出生时辰）
- 可选：双方长辈生肖（用于冲克提示）
- 硬排规则：日支正冲 / 杨公忌 / 三娘煞

### 命局先天分析
- 自动识别双方四柱本身的生克合冲关系
- 标记可"化解"的候选日期（命理上能缓和摩擦的选择）
- 分类展示：吉合、克制、冲突等关系

## 🚀 快速开始

### 无需安装
这是一个单页面 HTML 应用，下载 `wenqi.html` 后直接在浏览器打开即可使用。

```bash
# 方式一：本地打开
open wenqi.html

# 方式二：用任何 HTTP 服务器提供
python -m http.server 8000
# 然后访问 http://localhost:8000/wenqi.html
```

### 使用步骤

1. **输入双方信息**
   - 新娘的出生日期和时辰
   - 新郎的出生日期和时辰
   - （可选）双方长辈的生肖

2. **设置查询范围**
   - 选择预设范围（3个月、12个月等）
   - 或自定义起止日期

3. **配置择期规则**（高级设置）
   - 选择是否启用"日支正冲直接排除"
   - 选择是否启用"杨公忌/三娘煞作为硬排"
   - 选择是否启用"民俗附加扣分项"

4. **点击"开始排查吉日"**
   - 系统自动计算整个时间段内每一天的吉凶评分
   - 展示月历和榜单视图

5. **查看详细分析**
   - 点击任意日期卡片查看详细的分数拆解
   - 了解该日期为何得分高/低

## 🔮 核心概念

### 三层为何不能简单相加？
虽然表面上采用逐层加分制，但：
- **通书层** 反映的是"天时"（黄历法则）
- **生肖层** 反映的是"地利"（十二支的五行秩序）
- **八字层** 反映的是"人和"（个人命格的呼应）

三层之间存在关联性，不是独立的。比如：
- 日支正冲新郎生肖，既会在生肖层扣分，也可能在八字层产生冲克
- 但如果该日期的日柱能化解命主本身的缺陷，八字层的加分可能弥补掉生肖层的损失

### 关于"先天摩擦"
如果双方的四柱本身存在生克关系（如一方日干克另一方日干），工具会：
1. 在"双方命局先天关系"部分明确标出
2. 在候选日期中，标记那些日柱特别能缓和这种摩擦的选择（"✦化解"标签）

这**不是说这段婚姻不合适**——既然已经在择日，表明双方已决定缔结良缘。先天摩擦只是需要在后天通过择日来有意识地调和。

## ⚙️ 技术栈

- **前端框架**：纯 HTML + CSS + JavaScript（无依赖）
- **日期计算**：[lunar-javascript](https://github.com/6tail/lunar-javascript) 开源库
- **设计理念**：
  - 文言向的宋体 + 现代无衬线字体混排
  - 宣纸色系配色（暖米色背景）
  - 充分的留白和清晰的信息层级

## 📚 数据来源

- **通书规则**：取自流行的万年历和通书惯例
- **二十八宿、黄道黑道**：古代天文历法
- **杨公忌、三娘煞、大利月**：民俗口诀，各地流派记载略有出入
- **八字合婚理论**：紫微斗数 / 五行学说的通行解读

## ⚠️ 重要声明

1. **本工具仅供参考和娱乐**，不构成任何决策依据
2. **各流派说法不一**，择吉的学说众说纷纭，本工具采用的是较为常见的规则组合
3. **默认假设双方已决定缔结良缘**，不做"是否相配"的合婚判断
4. **八字层的计算** 仅比较候选日柱与双方四柱的呼应关系，并非完整的合婚分析
5. **隐私安全**：所有计算均在本地完成，无任何数据上传

## 🛠️ 自定义和扩展

如需修改评分规则、调整权重或增加新的择吉方法，可以编辑 `wenqi.html` 中的：

- `TIAN_SHE_GANZHI` 等常量定义
- `calcScores()` 函数中的具体加减逻辑
- CSS 变量（`--paper`、`--ink` 等）以改变配色方案
